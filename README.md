## 🔐 Random Password Generator

A simple yet powerful Python-based Random Password Generator that creates secure and unpredictable passwords using Python's built-in random and string modules. This project demonstrates the use of modules, string manipulation, functions, loops, and user input handling to solve a real-world security problem.

## 📌 Project Overview

The Random Password Generator allows users to generate strong passwords of their desired length. The generated password contains a combination of:

Uppercase Letters (A-Z)
Lowercase Letters (a-z)
Numbers (0-9)
Special Characters (!, @, #, $, %, etc.)

This tool helps users create secure passwords for online accounts and applications.

## ✨ Features

✔ Generate passwords of custom length
✔ Uses Python's built-in random module
✔ Uses Python's string module for character sets
✔ Strong and secure password generation
✔ User-friendly command-line interface
